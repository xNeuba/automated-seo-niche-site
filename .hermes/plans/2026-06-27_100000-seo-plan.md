# SEO Automatisierung: Initialer Projektplan

> **Ziel:** Aufbau eines vollautomatisierten SEO-Affiliate-Geschäfts mittels statischer Website und Gemini API.

## Architektur
- **Tech-Stack:** Astro (Static Site Generator), GitHub Actions (Automatisierung), Python (Content-Pipeline), Google Gemini API (Content).
- **Hosting:** Kostenloses Hosting via Vercel/Netlify.
- **Workflow:** 
  1. Automatisierte Keyword-Recherche.
  2. Generierung von Artikeln mittels Gemini API.
  3. CI/CD Pipeline (GitHub) baut und deployt die Seite.
